Principais formatações

Negrito / Destaque

Nesta frase temos um termo em negrito usando a tag B (Não Semântica)

Nesta frase temos um termo em destaque usando a tag STRONG (Semântica)

Itálico / Ênfase

nesta frase temos um termo em Itálico usando a tag I (Não Semântica)

Nesta frase temos um termo em ênfase usando a tag EM (Semântica)

Texto Marcado

Nesta frase temos um texto marcado usando a tag MARK (Semântica)

Texto grande e pequeno

Nesta frase temos um trecho grande usando a tag BIG (Não Semântica) e um trecho pequeno Usando a tag SMALL (Semântico)

Texto deletado

Também podemos marcar um texto excluído usando a tag DEL (Semântica)

Texto inserido

Podemos também marcar um texto como inserido usando a tag INS (Semântica)

Também podemos marcar um texto inserido usando a tag U (Não semântica)

Texto sobrescrito

Podemos escrever um texto sobrescrito (elevado) usando a tag SUP (Semântica), EX: x200+5=10

Texto subscrito

Podemos escrever um texto subscrito (baixo) usando a tag SUB (Semântica), EX: H2O, O3, CO2

Abreviações / Siglas

É possível criar abreviações usando a tag ABBR, como por exemplo com a sigla: SPFC
